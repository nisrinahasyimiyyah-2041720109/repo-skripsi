Dataset

Pada menu Dataset terdapat fitur untuk mengunggah file data ulasan dengan format .csv dan fitur scraping data dengan memasukkan link halaman website review Tokopedia yang kemudian ditampilkan dalam bentuk tabel.

Proses Data

Menu Proses Data digunakan untuk melakukan pemrosesan data (text preprocessing) agar siap untuk dilakukan prediksi sentimen menggunakan model yang telah dibuat sebelumnya.

Prediksi Sentimen

Proses prediksi sentimen terhadap file data ulasan yang telah diunggah dan melalui text preprocessing akan dilakukan pada menu ini. Sentimen akan dibagi menjadi tiga, yaitu negatif, netral, dan positif.
